## 说明

> 该日期控件包含常用功能,不依赖其它第三方库,浏览器需要支持ES5。

## 特性

1. 时间选择有两种模式:范围选择和单选
2. 月份切换
3. 最大时间和最小时间限制
4. 自定义皮肤
